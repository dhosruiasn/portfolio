# collage — 門後拼貼頁素材

門(BrandSection)打開後進入 GraphicOverlay，這裡放拼貼頁要用的圖。
無文字選單，純拼貼。

## 資料夾用途

| 資料夾 | 放什麼 | 格式 | 建議尺寸 |
|--------|--------|------|----------|
| `bg/`    | 底圖（黃色紙質＋顆粒質感，可含烤進去的黑白裝飾） | **JPG** | 3024 × 4200（桌機畫布 1512×2100 @2x）／手機 780 × 1688 |
| `items/` | 去背作品：海報、T-shirt、帽T、周邊等 | **PNG 透明底** | 各自原比例，長邊 ≥ 1000px（@2x） |
| `decor/` | 去背裝飾：黑白高對比小物、星星、貼紙徽章 | **PNG 透明底** | 依內容，@2x |

## 規則
- 底圖走 JPG，其餘全部去背 PNG，不要合成一張。
- 全部匯出 2×（這風格細節多，1× 會糊）。
- 檔名用英文小寫、連字號，例如 `poster-juliet.png`、`tee-besties.png`、`decor-camera.png`。
